# Gemini Studio - 电商图像生成工具

一款基于 Google Gemini AI 的高级电商产品摄影生成系统。

## 功能特点

✨ **AI 魔法扩写** - 自动将简短描述扩展为专业摄影指导

🎬 **场景裂变实验室** - 一张图生成 4 个相似但不同的场景变体

📱 **爆款文案生成** - 基于生成图片的小红书种草文案

🎨 **风格垫图** - 使用参考图片的光影和氛围

🔒 **主体锁定** - 精确控制商品主体保持不变

💾 **批量处理** - 同时处理多张图片

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 配置 API Key

访问 https://aistudio.google.com/app/apikey 获取您的 Google Gemini API Key

### 3. 启动应用

```bash
npm start
```

应用会自动打开 `http://localhost:3000`

## 使用指南

### 上传图片
- 在左侧"待处理队列"区域上传商品原图
- 支持批量上传多张图片

### 选择场景
- 系统会自动分析图片并推荐 3 套电商场景方案
- 也可以在环境预设中选择特定的背景风格

### AI 扩写提示词
- 输入简短的场景描述（如"木桌上的杯子"）
- 点击魔法棒🪄按钮，AI 会自动扩展为专业摄影描述

### 执行流水线
- 点击"开始执行流水线"按钮
- 系统会批量生成所有图片

### 导出结果
- 右侧矩阵中显示所有生成的图片
- 可以单独导出、预览、重绘或生成 HD 版本
- 点击"导出全部成品"下载所有结果

## 高级功能

### 风格垫图
上传一张参考图片作为光影和氛围的参考，AI 会模仿该图片的风格。

### 场景裂变
上传单张商品图，一键生成 4 个风格相似但细节不同的场景变体。

### 主体锁定
开启此模式确保商品主体 100% 保持不变，仅替换背景。

## 常见问题

**Q: 为什么生成失败？**
A: 检查您的 API Key 是否有效，或网络连接是否正常。

**Q: 如何获取 API Key？**
A: 访问 https://aistudio.google.com/app/apikey，使用 Google 账号登录创建。

**Q: 支持哪些图片格式？**
A: 支持 JPG、PNG 等常见格式，建议尺寸 800x800 以上。
